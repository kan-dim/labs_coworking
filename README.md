# Coworking
Автор - [Каневский Дмитрий](https://github.com/kan-dim)

Ключевая идея заключается в автоматизации добавления новых клиентов в существующем коворкинге, создание экскурсий, групп и так далле.

##  Client
В этом пакете написаны классы для описания клиента. У клиента есть два выборочных поля, это его основной язык и его тип. Всего у клиента могут быть два типа: фрилансер и представитель компании. ClientService служит для управления всеми клиентами.
##  Guide
В этом пакете создается реализация класса экскурсовода. Так как в будущем мы планируем запускать экскурсии, то для их проведения нам и понадобяться экскурсоводы. GuideService служит для их управления и для назначения свободного экскурсовода на новую экскурсию.
##  Manager
 Этот пакет создан организации работы менеджера. Менеджер (или же администратор) служит для управления всем коворкингом и решает все проблемы, если они возникают.
##  Excursion
 Этот пакет уже с более сложной реализацией. Он создан для осуществления экскурсий. Тут мы их создаем и управляем ими. Можем добавлять новые или удалять старые.
##  Group
Самый большой и сложный пакет. Тут мы для начала должны собрать в одну кучу уже существующих клиентов, потом подобрать для них уже готовую экскурсию с назначенным временим и экскурсоводом. В этом классе реализована валидация данных на основе регулярных выражениях.
##  Company
Здесь стартует программа. Класс Main создает и загружает в базу необходимые объекты, с которыми мы потом будем работать. Класс Menu необходим для реализации меню.
##  Room
Тут мы создаем один объект помещения, в котором и будут происходить все действия.

# Main
Модель объёмная и с большим количеством связей.
В программе доступен широкий функционал, а именно: 
1. Информация о компании

2. Добавить клиента
3. Удалить клиента
4. Список клиентов

5. Добавить менеджера
6. Удалить менеджера
7. Список менеджеров

8. Добавить экскурсовода
9. Удалить экскурсовода
10. Список экскурсоводов

11. Добавить экскурсию
12. Удалить экскурсию
13. Список экскурсий

14. Добавить группу
15. Удалить группу
16. Список групп

17. Завершить

Что позволяет гибко всем управлять.
